# 競合分析レポート：産後ケア出張サービス（東京都足立区）

**作成日：** 2026年2月27日
**対象事業：** 産後ケア検定資格保有看護師による出張産後ケアサービス
**対象エリア：** 東京都足立区

---

## 4. 参入ポジションの提案

### 4-1. ターゲット顧客層

**プライマリターゲット：**
- 足立区在住・産後0〜12か月の初産母親
- 行政産後ケア訪問型（最大7回）を使い切った後もサポートを必要とする母親
- 実家のサポートが受けられない孤立した産後ママ
- 産後の体調不良（傷の回復・貧血・産後うつ傾向）に不安を感じており、「医療職に診てもらいたい」ニーズがある母親

**セカンダリターゲット：**
- 外国にルーツを持つ産後ママ（足立区は外国人人口比率が高い）
- 多胎児（双子・三つ子）の産後ママ
- 産後2人目以降で上の子への対応も必要な産後ママ
- 産後1年超で行政補助対象外となった後もケアを求める母親

---

### 4-2. 競争優位性の源泉

#### 源泉1：「看護師×産後ケア検定」の組み合わせによる唯一性

足立区内の訪問型産後ケア事業者（ノア助産院・mammy助産院）はすべて**助産師**が主体であり、看護師が産後ケア検定を取得して訪問型サービスを提供する事業者は足立区内に事実上存在しない。

- 看護師は全国約180万人超（助産師の約60倍）存在し、地域へのアクセシビリティが高い
- 医療的視点（バイタル測定・創部観察・受診必要性の判断）を持ちながら、産後の生活支援・育児支援ができる
- 産後ケア検定による専門性で「一般看護師」との差別化も明確にできる

#### 源泉2：行政産後ケアの「7回上限後」を担うポジション

足立区の行政産後ケア訪問型は**最大7回**（宿泊・日帰り型との通算）の上限がある。7回の枠を使い切った後に「もう少しサポートがほしい」というニーズが必ず発生する。この行政サービスの「その後」を専門に担う民間サービスとしてポジションを取ることができる。

**連携戦略：**
- 行政訪問型の登録助産師（ノア助産院・mammy助産院）との連携・紹介ネットワーク構築
- 足立区子ども家庭支援センター・保健センターへの周知活動
- 保健師との協力関係を築き、ハイリスク産後ママ（孤立・産後うつ傾向等）の紹介を受ける

#### 源泉3：価格の「ちょうど良い」ゾーンの占有

| 価格ゾーン | 想定顧客 | 課題・空白 |
|-----------|----------|-----------|
| 2,000円（行政補助） | 低所得・非課税世帯 | 7回上限・助産師のみ |
| 8,000〜10,000円（自事業目標帯） | 中間層産後ママ | **本事業が担う空白地帯** |
| 15,000〜25,000円（既存民間高価格帯） | 高所得層 | 足立区の所得水準では利用困難 |

月2〜3回、1回8,000〜10,000円（交通費込み・60〜90分）という価格帯が足立区の中間層に受け入れられる現実的な水準と推定される。

#### 源泉4：アクセシビリティの最大化

- 訪問型は産褥期ママの「外出できない」という最大のバリアを解消する
- LINEでの予約受付・当日〜翌日の即日対応体制
- 足立区は外国にルーツを持つ住民が多いため、英語・中国語・韓国語等の多言語対応（簡単な日常会話レベルでも大きな差別化要因）

## 5. リスクと対策

| 価格抵抗 | 足立区の低所得水準による自費負担への心理的抵抗 | 行政7回利用後の「続き」として訴求。初回モニター価格（5,000〜6,000円）で実績と口コミを蓄積 |

## 6. 総括と推奨アクション

### 市場機会の評価

足立区の産後ケア市場は以下の理由から**参入余地が大きい**：

1. **供給不足：** 民間訪問型産後ケア事業者が2〜3社程度しかなく、競合密度が極めて低い
2. **行政サービスの構造的限界：** 最大7回という利用制限が明確な「その後のニーズ」を生む
3. **看護師資格の空白：** 既存競合はすべて助産師。看護師×産後ケア検定の組み合わせは足立区内で唯一無二
4. **高い潜在的支援ニーズ：** 低所得世帯が多く、実家サポートなしの孤立産後ママの割合が高い
5. **人口規模：** 足立区の総人口は約69万人と23区最大クラスで、出生数も相応に存在する

### 推奨ポジション

**「行政産後ケアの次のステップを担う、医療的視点を持った専門家」**

行政助産師の7回訪問を終えた後に「もう少しサポートしてほしい」というニーズに応える、看護師+産後ケア検定という資格の組み合わせのサービス。足立区の所得水準を踏まえた**1回8,000〜10,000円**（60〜90分、交通費込み）の価格帯で提供することで、競合が手薄な価格帯・サービスゾーンに参入できる。

### 優先アクション（立ち上げ90日）

| 優先度 | アクション | 目的 |
|--------|-----------|------|
| 高 | 足立区産後ケア事業の訪問型登録助産師（ノア助産院・mammy助産院）への挨拶・連携打診 | 紹介ネットワークの構築 |
| 高 | 足立区子ども家庭支援センター・保健センターへの事業説明と名刺配布 | 行政ルートからの認知獲得 |
| 高 | Instagram・LINE公式アカウント開設（「足立区 産後ケア 看護師」をキーワードに設定） | オンライン認知の確保 |
| 中 | 初回モニター価格（5,000〜6,000円/回）で5〜10名の実績事例を蓄積 | 口コミ・事例の創出 |
| 中 | 足立区内の産婦人科・クリニックへのサービス案内配布 | 紹介元の開拓 |
| 低 | 多言語対応（英語・中国語）の簡易案内チラシ作成 | 外国人産後ママ層へのアクセス |
