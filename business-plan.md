## 1. エグゼクティブサマリー

### 事業の一言コンセプト

> **「行政産後ケアの7回が終わっても、まだ終われない産後がある。足立区の産後看護師が、あなたの自宅に来ます。」**

### 事業の核心

本事業は、**産後ケア検定資格を持つ看護師が足立区内の産後母親の自宅を訪問し、医療的視点に基づく産後ケアを提供する完全出張型の個人事業**である。

足立区（東京23区内で平均世帯年収が最下位・約490万円）において、2025年度から訪問型産後ケアへの行政補助（1回2,000円、非課税世帯は無料）が新設された。しかし、この補助に対応できる民間訪問型事業者は事実上ゼロに近い状態であり、**巨大な供給不足**が生じている。

本事業はこの市場の空白を、**「看護師国家資格×産後ケア検定」という足立区内唯一の組み合わせ**で埋める。単なるサポートを超えた**医療的アセスメント能力**（バイタルサイン測定・産後うつスクリーニング・受診判断）と**行政補助申請サポート**という複合価値を提供し、足立区の中低所得層の産後母親にアクセスする。

## 2. 事業概要

### 2-1. 事業内容

**産後ケア検定資格を持つ看護師が、足立区内の産後母親の自宅を訪問し、医療的観点に基づく産後ケアを提供する完全出張型サービス。**

#### 提供するサービス内容

| カテゴリ | 具体的内容 |
|----------|-----------|
| **医療的アセスメント** | バイタルサイン測定（体温・血圧・脈拍）、産後出血・創部感染の早期察知・受診判断、産後うつスクリーニング（EPDS等） |
| **授乳・育児支援** | 授乳相談・ラッチオン指導、母乳マッサージ（乳房ケア）、育児相談・新生児ケア指導 |
| **産後メンタルケア** | 産後うつの傾聴・早期介入、育児疲労への精神的サポート、必要に応じた医療機関への橋渡し |
| **行政補助ナビゲーション** | 足立区産後ケア訪問型補助の申請サポート、利用可能な支援制度の案内（区の各種給付金・家事支援等） |
| **継続ケア** | 行政産後ケア7回終了後の継続ケア、産後1年超への対応 |

#### 事業の独自性（他サービスとの決定的な違い）

1. **看護師国家資格＋産後ケア検定**という組み合わせは足立区内に事実上存在しない
2. **行政補助申請サポートを付帯サービスとして提供**する訪問型産後ケア事業者は足立区内にゼロ
3. **行政産後ケアの「7回上限後」を専門に担う**民間サービスとしてのポジション

### 2-2. ビジョン・ミッション

**ミッション：** 「足立区の産後母親が、所得に関係なく、専門的な産後ケアに安心してアクセスできる社会を実現する」

**ビジョン（3年後の姿）：**
- 足立区内の保健センター3拠点全てから信頼・紹介される産後ケア専門家
- 月55件以上の稼働を維持する安定的な事業基盤
- 足立区の産後ケア補助制度の活用率向上に貢献する地域の専門家

### 4-3. 競争優位性の源泉

**源泉1：「看護師×産後ケア検定」の唯一性**
足立区内の訪問型産後ケア事業者はすべて助産師主体であり、看護師が産後ケア検定を取得して訪問型サービスを提供する事業者は事実上存在しない。

**源泉2：行政産後ケアの「7回上限後」を担うポジション**
行政の7回を使い切った後の継続ケアニーズに応える民間事業者として、行政サービスの**構造的補完者**としてのポジションを取る。

**源泉3：「ちょうど良い」価格ゾーンの占有**

| 価格ゾーン | 担い手 | 課題・空白 |
|-----------|--------|-----------|
| 2,000円（行政補助） | 行政委託助産師 | 7回上限・助産師のみ |
| **8,000〜10,000円（本事業目標帯）** | **本事業** | **担い手が存在しない空白地帯** |
| 15,000〜25,000円（民間高価格帯） | ノア助産院等 | 足立区の所得水準では利用困難 |

**源泉4：補助申請サポートという情報格差の解消**
申請手続きが複雑な区の補助制度において、「申請代行サポート付き」という独自価値を提供することで、申請困難な低所得層・外国籍母親へのアクセスを最大化する。

1. **足立区産後ケア訪問型指定事業者への登録**を最優先で進め、公的補助の枠組みを活用

## 7. マーケティング・顧客獲得戦略

### 7-1. 戦略の全体像

足立区の顧客の最重要認知経路は**行政接触ポイント（保健師・こども家庭センター）**であり、SNSはその補完的な役割を担う。

```
【優先順位】
1位：行政連携（保健センター・産婦人科等）
2位：SNS・LINE（Instagram・LINE公式）
3位：地域オフライン（都営住宅・産院チラシ）
```

### 7-2. 行政連携戦略（最重要）

#### 優先連携機関

| 機関名 | 場所 | 連携内容 |
|--------|------|---------|
| **千住保健総合センター** | 千住仲町19-3 | 妊婦健診・産後訪問の紹介先登録申請 |
| **竹の塚保健総合センター** | 西竹の塚1-11-2 | 竹の塚・西新井・綾瀬エリアをカバー |
| **江北保健総合センター** | 西新井本町2-30-40 | 区北部エリア担当 |
| **足立区こども家庭センター** | 区内（令和7年度設置） | 妊娠期〜子育て期の一体的支援拠点 |
| **区内産婦人科6〜8院** | 区内各所 | 退院直後の母親へのサービス案内 |

#### 行政連携のポイント

1. **区の訪問型産後ケア指定事業者として早期登録** → 「第1号事業者」としての行政側へのアピール
2. **「こんにちは赤ちゃん訪問」後の継続ケア先**として保健師の紹介リストへの掲載を目指す
3. **2025年10月開始「子育て全世帯訪問事業」**への連携申請・紹介先登録を最重点目標に設定

### 7-3. SNS・デジタル戦略

#### Instagram（メイン集客チャネル）

- **ローカルハッシュタグ：** `#足立区ママ` `#足立区子育て` `#綾瀬ママ` `#竹の塚ママ` `#千住ママ` `#西新井ママ`
- **コンテンツ方針：** 「足立区で使える産後ケア助成金・補助まとめ」シリーズ投稿（保存率・シェア率が高い）
- **顔出し投稿：** 看護師スタッフの資格・顔出し紹介（信頼感醸成）

#### LINE公式アカウント（予約・フォロー用）

- 予約・問い合わせ窓口をLINEに一本化
- **「補助申請のサポートもします」**メッセージでの差別化
- 「今週の育児ヒント」などの週1配信で関係維持

### 7-4. 地域密着型オフライン施策

| 施策 | 対象場所 | 内容 |
|------|---------|------|
| チラシ設置 | 区内産婦人科6〜8院 | 「退院後のお母さんへ」向けA5チラシ |
| 掲示板活用 | **都営住宅・区営住宅共用部** | **「住民税非課税世帯は自己負担なし」を明示** |
| チラシ設置 | 認定こども園・保育園 | 「2人目のお母さんへ」メッセージ |

### 7-5. 補助申請サポートという差別化の核

足立区の訪問型補助は事前申請・承認通知書の取得が必要で手続きが複雑。この手間を代行またはサポートするサービスを「**無料付帯**」として提供することで、申請困難な低所得層・外国籍母親へのアクセスを最大化する。

### 8-2. 損益分岐点

```
月間固定費          = 45,000円
1件あたり変動費     = 500円
1件あたり平均売上   = 4,800円
1件あたり限界利益   = 4,800円 - 500円 = 4,300円

損益分岐点件数      = 45,000円 ÷ 4,300円 ≒ 10.5件
→ 月間11件以上の訪問で黒字化（稼働率18%という低い目標）
```

リスク対応追加後の損益分岐点：**月13〜14件**（依然として低水準）

## 9. リスクと対策

### 9-1. 主要リスクの優先順位（足立区特化）

| 優先度 | リスク | 対応策の要点 |
|--------|--------|-----------|
| **1位** | **低単価による収益性の慢性的な低さ** | 月額パッケージ化・段階的値上げ・オンライン相談の副収入化 |
| **2位** | **行政補助制度の変更・縮小** | 自費プランの比率を3年目60%以上へ段階的に引き上げ |
| **3位** | **外国人言語バリア（足立区固有）** | 多言語資料・翻訳アプリの開業前整備。区外国人相談窓口との連携 |
| **4位** | **行政連携構築の難易度** | 開業90日以内に保健センター3拠点を訪問 |
| **5位** | **移動コスト・時間コスト（広域区）** | エリアを3ゾーンに分割・電動アシスト自転車の導入 |

### 9-2. 足立区固有リスク（5項目）

| リスク | 発生確率 | 対応策 |
|--------|---------|--------|
| 外国人言語バリア | **高** | 英語・中国語・韓国語・ベトナム語の簡易資料作成。翻訳アプリをタブレットに常備 |
| 訪問時の安全（DV家庭等） | 低〜中 | 初回訪問前の電話確認・行動記録の第三者共有・DV対応プロトコル準備 |
| 低所得層の支払いトラブル | 低〜中 | PayPay等による事前決済の原則化・料金を書面で事前確認 |
| 移動コスト・時間コスト | 中 | 足立区を北部・中部・南部3ゾーンに分割し曜日ごとにエリア固定 |
| 行政連携構築の難易度 | 中 | 「競合」ではなく「補完的パートナー」としてノア助産院・mammy助産院に接触 |

### 9-3. 行政補助廃止シナリオの財務影響

廃止時の損益分岐点変化：月11件 → **月16〜18件**

緊急対応策：
1. 自費プランを月額パッケージ（17,600円/月）に集約し継続顧客を固定化
2. オンライン産後相談（2,000〜3,000円/30分）を新収益源として立ち上げ
3. 隣接区（葛飾区・荒川区）への訪問エリア拡大
