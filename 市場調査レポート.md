# 市場調査レポート
## 産後ケア出張サービス（東京都世田谷区）

**調査日：** 2026年2月27日
**対象事業：** 産後ケア検定資格を持った看護師による出張産後ケアサービス
**対象地域：** 東京都世田谷区

---

## エグゼクティブサマリー

本レポートは、世田谷区を対象とした産後ケア出張サービス事業への参入可能性を評価するための市場調査結果をまとめたものである。産後ケア市場は、政府の少子化対策「こども未来戦略」による制度的後押しと社会的関心の高まりを背景に急速に拡大しつつある。世田谷区は東京23区最大の人口規模（約92万人）と高い所得水準、年間約6,000件の出生数を持ち、出張型産後ケアサービスの有望な参入先と判断される。一方で、公的サービスの認知不足による利用率の低さ（24.7%）は、質の高い民間出張サービスにとって未充足ニーズが大きいことを示している。

---

## 1. 市場規模の調査

### 1.1 国内産後ケア市場の規模

国内産後ケア市場の独立した統計数値は限られているが、以下のデータが参考となる。

| 指標 | 数値 | 出典 |
|------|------|------|
| こども家庭庁 産後ケア事業予算（令和6年度） | 約60.5億円 | こども家庭庁 |
| 令和7年度母子保健対策関係予算 | 176.35億円 | こども家庭庁 |
| 産後ケア事業実施自治体数（2024年度末） | 1,644自治体（実施率94%） | こども家庭庁 |
| 産後ケア事業 産婦利用率（2024年度） | 24.7% | こども家庭庁 |
| 同（2023年度） | 15.8% | こども家庭庁 |

- 世界マタニティケア市場は2022年に約5,500億米ドル規模であり、2035年までに約1兆8,500億米ドルへ拡大が予測されている（The Insight Partners）
- 日本の産後ケア市場は、公的補助の拡充・民間サービス増加を背景に拡大局面にある
- 民間産後ケア施設（ホテル型・訪問型）の市場は独立した統計がないが、実施自治体数が2020年度の約900から2024年度の1,644へと83%増加しており、市場の急速な拡大が示される

### 1.2 訪問型（アウトリーチ型）産後ケアの位置づけ

産後ケア事業は3形態に分類される。

- **宿泊型（ショートステイ）：** 産院・助産院等に宿泊し支援を受ける形態
- **デイサービス型（通所型）：** 施設に日帰りで来所する形態
- **アウトリーチ型（訪問型）：** 支援者が利用者宅に訪問する形態

訪問型は、移動負担がない・乳児を連れて外出する必要がないという点で潜在的ニーズが高く、公的サービスの供給不足が顕著な形態である。世田谷区立産後ケアセンターでは訪問型の利用を受け付けているが、利用上限は通算7回（デイケアとの合算）と限られている。

### 1.3 世田谷区の出生数・対象人口

| 指標 | 数値 |
|------|------|
| 世田谷区総人口（2025年1月時点） | 約92.3万人 |
| 年間出生数（2024年） | 5,893人 |
| 年間出生数（2023年） | 5,919人 |
| 人口千人当たり出生率（2024年） | 6.59人（全国平均5.68人を上回る） |
| 20〜39歳人口の割合 | 総人口の13.9%（全国平均10.3%を上回る） |

- 2024年の出生数は9年連続の減少であるものの、全国平均を上回る出生率を維持しており、東京23区の中でも最多水準の出生数を誇る
- 年間約6,000人の新生児は、直接的なサービス対象（産後1年以内の母親）として継続的に創出される市場規模を示す

## 2. 市場トレンドの調査

### 2.1 産後ケアへの社会的関心の高まり

- 産後うつの認知は近年急速に広がっており、20〜30代女性の72%が「産後ケアを知っている」と回答（同世代全体では52%）
- 産後に悩みやトラブルを経験した保護者は9割以上であり、ニーズは潜在的に非常に高い
- 利用経験者の9割が「産後ケアをおすすめしたい」と回答しており、満足度は高い
- 20〜30代女性の67%が「産後ケアを利用したい」と回答しており、サービスへの意欲は明確である（BABY JOB株式会社調査）

### 2.2 行政の支援制度（産後ケア事業）

**国の動向**
- 2019年母子保健法改正により、産後ケア事業が市区町村の努力義務として法制化
- 2024年子ども・子育て支援法改正により、産後ケア事業が「地域子育て支援サービス」として位置づけ強化
- 2024年度より、自治体への補助金の上限撤廃（日本経済新聞報道）
- 2024年度より、全世帯への利用料軽減措置を拡大

**世田谷区の取り組み**
- 世田谷区立産後ケアセンター（公益社団法人日本助産師会運営受託）が存在
- アウトリーチ（訪問型）産後ケアを実施：3,000円/回（クーポン利用で500円）
- 「産後ケア事業クーポン」「せたがや子育て利用券」の使用が可能
- 利用上限：デイケアとの合算で最大7回まで

**世田谷区の費用補助スキーム（訪問型）**
| 区分 | 利用者負担額 |
|------|------------|
| 通常（クーポンなし） | 3,000円/回 |
| クーポン利用（5日上限） | 500円/回 |
| 非課税世帯等 | 減免制度あり |

### 2.3 少子化対策としての政策動向

- 2023年12月閣議決定「こども未来戦略」において、3.6兆円規模の「こども・子育て支援加速化プラン」を策定
- こども家庭庁予算の倍増を目標として掲げており、産後ケア関連施策への財政投入が継続拡大している
- 2025年4月から「妊婦等包括相談支援事業」「妊婦のための支援給付」が法制度として開始
- 産後ケア事業は2025年度より「子育て支援交付金」事業群に統合され、都道府県の財政的関与も明確化

### 2.4 コロナ後の在宅サービスへの需要変化

- コロナ禍でテレワークが普及し、子育て中の在宅勤務が一般化。産後の母親が「自宅にいながら専門的ケアを受けたい」という需要が顕在化
- コロナ禍で緊急事態宣言下においても、60%以上の保護者が「在宅中の育児支援サービスを利用したい」と回答
- コロナ収束後も在宅勤務を継続する女性は7割を超えており（日本経済新聞）、自宅でのサービス受給への親和性が高い
- 外出制限を経験した母親世代は「自宅に来てくれるサービス」への抵抗感が低下している

### 3.1 世田谷区の人口統計

|------|------|

| 20〜39歳人口比率 | 13.9%（全国平均10.3%を大幅に上回る） |

- 20〜39歳人口：約128,000人（推計）
- 産後ケアの直接ターゲット（出産後1年以内の母親）：年間約5,900人
- 出産前から含む潜在顧客（妊産婦全体）：年間約6,000〜7,000人

### 3.2 世田谷区の平均世帯収入

| 指標 | 数値 |
|------|------|
| 平均年収 | 655万円（東京都内5位） |
| 平均所得（2023年） | 619万8,160円（全国市区町村ランキング10位） |
| 年収1,000万円以上の世帯比率 | 28.5%（東京都内6位） |
| 単身世帯年間支出合計 | 264万円（東京都内7位） |

- 高所得者層が多く集住しており、1万〜2万円程度の民間サービス利用料を自己負担できる経済力を持つ世帯が多い
- 年収300万円未満の世帯も26.8%存在するが、行政補助との組み合わせによる複層的な価格設計が有効

### 3.3 産後ケアサービスへの支払い意欲

民間産後ケアサービスの一般的な価格帯（参考値）：

| サービス形態 | 市場価格帯 |
|------------|-----------|
| 宿泊型（民間産後ケアホテル） | 1泊あたり2万〜10万円 |
| 日帰り型（民間施設） | 1回あたり1万〜3万円 |
| 訪問型（公的事業・補助前） | 1回あたり3,000〜10,000円 |
| 訪問型（民間・専門職派遣） | 1回あたり8,000〜20,000円（推定） |

- 世田谷区の高所得層（年収1,000万円以上28.5%）は、月2〜4回の出張訪問ケア（1回1〜2万円）を継続利用できる経済力を持つ
- 産後ケアリスト1級による商業サービスは、訪問1回あたり1万〜1.5万円程度が相場水準と推定される
- 産後ケア需要のある母親のうち、実際に利用した割合は約3割であり、残り7割が未利用の潜在市場を構成

## 4. 参入機会の特定

### 4.1 市場のギャップ・未充足ニーズ

**ギャップ1：公的サービスの利用上限不足**
- 世田谷区の公的訪問型産後ケアは最大7回（デイケア合算）が上限。産後3〜6ヶ月の支援ニーズに対し、回数が不足している
- 民間サービスにより上限なしの継続的なサポートを提供できる余地が大きい

**ギャップ2：認知・アクセスのギャップ**
- 産後ケアを「知らない」20〜30代女性が45.5%存在する
- 利用しない最大の理由が「利用の仕方がわからない」である
- 直接的なマーケティング（SNS、産婦人科連携、母親学級連携）で差別化が可能

**ギャップ3：移動困難の解消**
- 施設型サービスでは「乳児を連れて移動することの大変さ」が障壁になっている
- 出張型サービスはこの障壁を根本から解決し、明確な差別化優位性を持つ

**ギャップ4：専門職（看護師）による信頼性**
- 現在の民間産後ケアサービスは産後ケアリストや助産師が担うケースが多い
- 産後ケア検定資格を持った看護師による訪問は、医療的背景に基づく信頼性・専門性で差別化できる
- 乳腺炎、産後の体調管理、創傷ケアなどへの対応が可能であることは大きな競争優位となる

**ギャップ5：夜間・週末対応の不足**
- 公的産後ケアは平日昼間中心
- 共働き家庭（特に夫が帰宅する夕方以降や週末）に対応するサービスのニーズが高い

### 4.2 成長ドライバー

| 成長ドライバー | 内容 |
|-------------|------|
| 政策的追い風 | こども未来戦略3.6兆円、産後ケア事業予算の継続的拡大 |
| 市場ニーズの顕在化 | 産後うつ・育児不安への社会的関心の高まり |
| 高所得地域の特性 | 世田谷区の高所得層（年収1,000万円超28.5%）による購買力 |
| 少子化危機感の高まり | 2024年出生数過去最少水準、社会全体での育児支援強化の機運 |
| 在宅勤務定着 | コロナ後も在宅勤務継続の女性が多く、自宅サービスへの親和性が向上 |
| 専門職の稀少性 | 産後ケア検定資格+看護師という専門職の組み合わせは市場に少ない |
| 行政との連携機会 | 自治体の委託先・補助対象事業者として認定される可能性 |
| 口コミ効果 | 利用経験者の9割が「おすすめしたい」と回答。SNSを通じた口コミ拡散が期待できる |

## 6. 結論と戦略的示唆

### 6.1 市場参入の評価

| 評価項目 | 評価 | 理由 |
|---------|------|------|
| 市場規模 | ◎ | 年間5,900人の出生数、高い潜在需要 |
| 市場成長性 | ◎ | 政策的追い風・社会的関心の急拡大 |
| 競合環境 | ○ | 参入者は増えているが、専門看護師資格特化は差別化余地大 |
| 顧客の支払意欲 | ◎ | 高所得地域。民間サービス単価への抵抗が相対的に低い |
| 参入障壁 | ○ | 産後ケア検定+看護師免許は高い参入障壁となる |
| リスク | △ | 競合の増加・行政サービスの充実による代替リスク |

### 6.2 重点推奨戦略

1. **専門性の徹底訴求：** 産後ケア検定資格+看護師免許による「医療バックグラウンドのある産後ケア」を差別化軸とする
2. **世田谷区行政との連携：** 区の委託先認定や「せたがや子育て利用券」対象事業者登録を目指し、公的補助と組み合わせた価格体系を構築する
3. **紹介・SNS重視のマーケティング：** 満足度の高い利用者による口コミ（利用者の9割が推薦）をSNSで拡散する施策を初期から組み込む
4. **産婦人科・助産院との提携：** 退院後のケアとして紹介を受ける体制を構築し、ターゲット顧客との接点を確保する
5. **複層的な価格設計：** 行政補助対象のベーシックプランから、高所得層向けの充実プランまで設定し、市場全体をカバーする
