# 第1章 エグゼクティブサマリー

## 事業の概要

本事業は、「産後ケア検定資格を持った看護師」が世田谷区内の産後母親の自宅に出張し、母体ケア・授乳サポート・新生児ケア・メンタルサポートを提供するプレミアム産後ケアサービスである。

**ビジョン：**
> 「世田谷区の産後に、医療の安心を。」
> 世田谷区の"かかりつけ産後ケア看護師"として、産後の母親が最も孤独で脆弱なときに、医療の安心感と人間的なぬくもりを自宅で届ける。

**メインキャッチコピー：**
> 「看護師だから、気づける。産後のあなたに、本当のケアを。」

## 市場機会の要約

| 指標 | 数値 |
|------|------|
| 世田谷区年間出生数（2024年） | 5,893人（東京23区最大水準） |
| 産後ケア利用率（2024年度） | 24.7%（前年比+9pt、急拡大中） |
| 産後ケアを「利用したかった」母親 | 6割以上（潜在需要が大きい） |
| 世田谷区平均世帯年収 | 655万円（年収1,000万円超 28.5%） |
| 政策予算（令和7年度母子保健） | 176億円（継続的拡大） |

- 公的訪問型産後ケアは**抽選制・平日限定・最大7回**という供給制約がある
- 民間競合の大半は**医療資格なし**のドゥーラや家事代行
- 「看護師×産後ケア検定」による医療品質の訪問専門サービスは**市場空白地帯**

## 4-2. 競争優位性（3本柱）

### 柱1：資格による模倣困難な参入障壁

- **看護師国家資格 + 産後ケア検定** の組み合わせは市場で希少
- 乳腺炎・会陰傷・産後うつ等の医療的異常を早期発見できる唯一のポジション
- 競合のドゥーラ・家事代行サービスには絶対に提供できない医療的判断が可能
- 訴求メッセージ：「医師への相談が必要なサインを早期発見できる、看護師だからできるケア」

### 柱2：世田谷区特化の地域密着戦略

- 世田谷区の助成制度（産後ケアクーポン12,500円分）に完全対応
- 区内産婦人科・助産院・保健センターとの連携ネットワーク構築
- 区立産後ケアセンターの「待機・抽選漏れ」母親への補完的サービス
- 「世田谷区のかかりつけ産後ケア看護師」としてのブランド確立

### 柱3：柔軟なサービス設計

| 競合の弱点 | 対応するサービス設計 |
|-----------|------------------|
| 公的センターの平日・時間帯固定 | 土日・夜間対応可能 |
| 産後ヘルパーの高価格 | 看護師プレミアム付きで合理的な中価格帯 |
| ドゥーラの料金不透明 | 明瞭な料金体系（時間単位） |
| 施設型の移動負担 | 完全訪問型（外出不要） |

## 4-3. 持続的競争優位（SCA）の3層構造

```
【第1層：模倣困難な資格的参入障壁】（最も強固・長期的）
  看護師国家資格 × 産後ケア検定
  → 誰でも名乗れない。取得に数年を要する

【第2層：経験の蓄積による暗黙知の優位】（中期的）
  初期顧客への対応から蓄積される産後ケアのノウハウ
  地域のネットワーク（産婦人科・助産院・行政との関係）

【第3層：口コミ・ブランドによるネットワーク効果】（中長期的）
  利用経験者の9割が「おすすめしたい」と回答
  先行者が圧倒的に有利な口コミ伝播
```

## 8-2. 重点リスクへの対応策

### ①人材確保困難（B-1）
- 開業3〜6ヶ月前から産後ケア検定コミュニティへの参加
- 看護師転職サイトへの登録・業務委託候補者のパイプライン構築
- 産後ケア検定取得支援制度（受験費用補助）の整備

### ②訪問中の事故（O-1）
- 訪問前・中・後の標準業務手順書（SOP）整備
- 賠償責任保険（1事故5,000万円・年間補償1億円）に加入
- 初回訪問時に顧客へのリスク説明・同意書の取得

### ③キャッシュ枯渇（F-1）
- 緊急予備資金200万円を運転資金とは別口座で管理
- 月次でKPIと財務モデルを照合、2ヶ月連続乖離で対策発動
- セーフティネット融資の借入枠を開業後6ヶ月以内に確保

### ④偽装請負リスク（L-3）
- 業務委託契約書を社会保険労務士が作成・監修
- 指揮命令関係が生じないよう業務内容・時間・場所の自由裁量を明記
- 年1回の定期的な契約レビューを実施
