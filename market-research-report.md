## 2. 市場トレンドの調査

### 2-1. 産後ケアへの社会的関心の高まり

- **政策的後押し**: 2021年に母子保健法改正で産後ケア事業が法定化・努力義務化。2023年には国費による利用者負担の軽減措置（2,500円/回、上限5回）が全産婦を対象に拡大
- **少子化対策の最前線**: 「こども未来戦略」（2023年閣議決定）において産後ケアは少子化対策の重要施策として明示。3.6兆円規模の「こども・子育て支援加速化プラン」が策定されている
- **孤立した産後女性への支援ニーズ**: 核家族化・地縁の希薄化を背景に、産後のサポート不足が社会課題として認識されている
- **補助制度が利用を後押し**: 産後ケア利用者の53.85%が「自治体の補助があったから利用した」と回答（BABY JOB株式会社調査）
- **高い満足度とクチコミ効果**: 産後ケア事業利用者の9割が「周囲にすすめたい」と回答

### 2-2. 足立区の産後ケア行政補助制度（令和7年度時点）

| 項目 | 内容 |
|------|------|
| 補助対象者（訪問型） | 区内在住で産後1年未満の母親と乳児 |
| 申請可能時期 | 妊娠28週以降（オンライン申請または窓口申請） |
| 利用回数（上限） | 宿泊型・日帰り型・訪問型を合わせて**7回（7日）** |
| **訪問型 利用者負担** | **1回あたり2,000円**（住民税非課税世帯は無料） |
| **訪問型開始時期** | **令和7年度（2025年度）から新規拡充** |
| 既存施設の補助（宿泊型） | 1日5,000円（日帰り型は1日2,500円）、生活保護・非課税世帯は減免 |
| 対象施設（訪問型） | ノア助産院など区指定の事業者 |

**重要**: 令和7年度（2025年度）から訪問型産後ケアが足立区の補助事業として新設された。指定事業者として登録することで、利用者の自己負担を1回2,000円（非課税世帯はゼロ）に抑えることができ、参入障壁の低下と需要拡大が期待される。

### 2-3. 足立区の子育て支援施策（2024〜2025年）

| 施策名 | 内容 |
|------|------|
| あだちスマイルママ＆エンジェルプロジェクト | 妊娠期〜子育て期の切れ目のない支援体制（保健師による相談等） |
| あだち出産・子育て応援給付金 | 妊婦への経済的支援給付（妊婦1人あたり5万円相当） |
| 産前・産後家事支援事業 | ヘルパー派遣による家事支援（産前産後の生活支援） |
| 子育て全世帯訪問事業（2025年10月開始） | 生後5ヶ月〜1歳4ヶ月の全世帯へ保育士等が1〜2ヶ月ごとに訪問 |
| 待機児童解消 | 令和6年4月1日現在の待機児童5人（大幅解消済み） |
| あだちマイ保育園 | 保育施設非利用家庭への保育士による相談支援 |
| 入学準備金（2026年〜） | 小中入学時に1人10万円支給（東京23区最大規模・総額18.7億円） |

## 3. ターゲット市場の特性

### 3-1. 足立区の人口統計

| 指標 | 数値 | 出典・備考 |
|------|------|------|
| 総人口（令和6年7月） | 約69万人 | 東京23区内で人口第2位の大規模区 |
| 年間出生数（2024年） | 3,815人 | 9年ぶりの増加（前年比+1.0%） |
| 産後ケア年間対象者（推計） | 約3,800〜4,000人 | 当年出生数ベース |
| 保育所等待機児童数（令和6年4月） | 5人 | 大幅に解消済み、共働き世帯を支える基盤整備が進む |
| 共働き世帯比率 | 増加傾向 | 保育施設の充実が後押し |

### 3-2. 足立区の平均世帯収入（世田谷区との比較）

| 区 | 平均世帯年収 | 東京23区内順位 | 備考 |
|------|------|------|------|
| **足立区** | **490万円** | **最下位（23位）** | 東京都平均564万円より74万円低い |
| 世田谷区 | **655万円** | 上位 | 東京23区内でも高所得区 |
| **差額** | **▲165万円** | — | 足立区は世田谷区より年間165万円低い |

**年収階級別世帯分布（足立区）**:

| 年収階級 | 世帯比率 | 産後ケア価格感度 |
|------|------|------|
| **300万円未満** | **37.0%（最多）** | 非常に高い → 区補助（非課税世帯：自己負担ゼロ）が必須 |
| 300万円〜500万円未満 | 29.1% | 高い → 区補助（自己負担2,000円）が利用のカギ |
| 500万円〜700万円未満 | 17.0% | 中程度 → 補助＋追加サービスの組み合わせが有効 |
| 700万円〜1,000万円未満 | 10.8% | やや低い → 自費の付加サービスも検討可能 |
| 1,000万円以上 | 約6.1% | 低い → 自費の高付加価値サービスにアクセス可能 |

**示唆**: 足立区は東京23区内で最も平均世帯年収が低い区（490万円）であり、世田谷区（655万円）との差は約165万円に上る。低〜中所得層が全世帯の約66%を占めるため、**区の補助制度（自己負担2,000円/回、非課税世帯はゼロ）の活用が利用率向上の最大の鍵**となる。

### 3-3. 産後ケアサービスへの支払い意欲と価格感度

**価格感度の特性**:
- 足立区の平均世帯年収490万円という水準は、民間産後ケアの自費サービスに対する価格感度が高いことを示す
- 産後ケアホテルに関する調査では、利用しない最大理由は「料金が高いから」（全国調査）
- 「住民税非課税世帯」が多い足立区では、区補助による**自己負担ゼロ**が最大のアクセス障壁除去となる

**支払い意欲の特性**:
- 区の補助制度を活用し自己負担を2,000円/回に抑えた場合、利用意欲は大幅に高まると予測される
- 「補助があったから利用した」が53.85%（全国調査）という傾向は、足立区においてより顕著に現れると推測される
- 利用した産婦の9割が満足し「周囲にすすめたい」とするクチコミ効果が獲得コストを削減する

**推奨価格帯の考え方**:

| 区分 | 対象世帯 | 価格帯 | 備考 |
|------|------|------|------|
| 区補助対象（訪問型） | 産後1年未満・全世帯 | **2,000円/回**（非課税は0円） | 指定事業者登録が前提 |
| 区補助対象外（追加サービス） | 中〜高所得層向け | **5,000〜10,000円/回** | 授乳指導・産後メンタルケア等 |
| 完全自費サービス | 高所得層（1,000万円以上） | **10,000〜15,000円/回** | 高付加価値・専門性訴求 |

## 4. 参入機会の特定

### 4-1. 足立区特有の市場ギャップ・未充足ニーズ

| 市場ギャップ | 内容 |
|------|------|
| **訪問型の供給不足** | 令和7年度から区補助が開始されたが、対応できる指定事業者は現時点でノア助産院などごく少数。需要増加に対して供給が大幅に不足している。 |
| **看護師資格の希少性** | 足立区内の訪問型産後ケアは主に助産師が担っている。「産後ケア検定資格を持つ看護師」は医療的知識に裏打ちされた安心感を提供でき、明確な差別化ポイントになる。 |
| **低所得層への支援の空白** | 300万円未満世帯が37%を占める。これらの世帯は区補助（非課税世帯は自己負担ゼロ）の対象だが、情報格差により制度へのアクセスができていないケースが多い。 |
| **産後うつ・孤立リスク層へのアウトリーチ** | 核家族・単身移住者が多い足立区では産後の社会的孤立リスクが高い。訪問型は通院困難な産婦への唯一のアクセス手段として機能できる。 |
| **補助制度の認知不足** | 産後ケア事業の利用率は全国的に低い（10.9%）。地域密着型のマーケティングにより、制度未認知の母親を新規利用者として開拓できる余地が大きい。 |
| **競合の少なさ** | キッズラインなどのプラットフォーム型は存在するが、足立区内で「看護師資格＋産後ケア検定」を持つ専門的出張サービスのプレーヤーは確認されていない。 |

## 5. 総合評価とまとめ

### 市場魅力度サマリー

| 評価項目 | 評価 | コメント |
|------|------|------|
| 市場成長性 | ★★★★★ | 産後ケア需要は政策・社会的背景から継続的拡大 |
| 制度的追い風 | ★★★★★ | 令和7年度から区の訪問型補助が新設。参入タイミングは最適 |
| 競合環境 | ★★★★☆ | 訪問型専門事業者は少なく、参入余地が大きい |
| 収益性 | ★★★☆☆ | 区補助内では自己負担2,000円。追加・自費サービス設計で収益確保が必要 |
| 価格感度リスク | ★★★☆☆（中程度） | 低所得層が多いが補助制度で緩和可能。中高所得層への付加価値訴求も重要 |
| ターゲット接触性 | ★★★★☆ | 区の子育て支援ネットワーク・産院との連携で集客経路を確保できる |

### 戦略的推奨事項

1. **令和7年度（2025年度）の足立区産後ケア訪問型指定事業者への登録を最優先で進める**。これにより利用者自己負担が2,000円/回（非課税世帯はゼロ）に抑えられ、特に低所得層へのアクセスが確保される。

2. **看護師資格＋産後ケア検定による差別化訴求**を明確にする。医療的安全性・専門性は、助産師・産後ドゥーラとの競合から差別化できる核心的優位性となる。

3. **区補助対象の7回利用を基本パッケージとしつつ、追加の自費サービス（産後メンタルケア、授乳相談、育児指導等）を組み合わせた複合サービスモデル**を構築し、収益性を高める。

4. **足立区の子育て支援センター・産院・保健センターとの連携紹介ルートを早期に構築する**。区が2025年10月から全乳幼児世帯訪問事業を開始するため、その紹介先として位置付けられる可能性もある。

5. **年収300万円未満世帯（37%）へのアウトリーチ**には、住民税非課税世帯の自己負担ゼロ制度を積極的に周知する。情報格差が最大のアクセス障壁であり、区の広報・子育て支援サロンでの周知活動が有効。

6. **足立区は東京23区で最も平均世帯年収が低い区（490万円、世田谷区655万円より165万円低い）**という現実を踏まえ、価格設計は区補助制度を中心に据え、付加価値型自費サービスを副次的に位置付ける戦略が適切。
